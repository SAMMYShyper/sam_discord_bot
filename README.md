# sam_discord_bot
This is my discord bot. Inspired by Indently, FreeCodeCamp, and many other sources. Currently contains funny features (affirmations, quotes, coin flips, and !help) and OpenAi's GPT API. 

## To use the bot
- download
- keep init files empty
- create a file called ".env" and write two lines:
  - SAMSTOKEN = Your discord developer token
  - GPTKEY = Your API key for using gpt
 - feel free to change the name from sams token to discord or something more intuitive 
- Once this is ready, pip install all the libraries you lack, indicated by a squiggly red line if not installed: (dotenv, discord, openai, etc.) 
- Finally, run using the `run.py` file
